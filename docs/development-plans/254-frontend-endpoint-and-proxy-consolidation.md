## 2. 交付物
- 端点/代理配置说明与最小示例（仅引用前端配置与 .env/.npmrc）；
- Playwright 运行约定（PW_BASE_URL、PW_SKIP_SERVER 等）与示例命令；
- 证据登记：logs/plan254/*（前端端点连通与代理验证输出）。

## 3. 验收标准
- 本地开发（vite 或 webServer）与 CI 中 Playwright 行为一致；
- 端点与代理配置的单一事实来源清晰（不复制，仅索引）。
