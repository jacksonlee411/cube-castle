# PeopleSoft Core HR 功能菜单参考

**文档编号**: 79
**创建日期**: 2025-10-13
**当前版本**: 2.0
**文档类型**: 参考资料

## 概述

本文档记录 PeopleSoft Core HR（核心人力资源）的完整功能菜单结构，涵盖 21 个功能模块，包括：

- **基础运营模块**（1-10）：组织、人员、职位、人事、工作信息、薪酬、福利、时间考勤、自助服务、报表分析
- **人才管理模块**（11-14）：招聘、绩效、培训、人才管理
- **薪资与合规模块**（15-16）：薪资计算、合规管理
- **专项管理模块**（17-20）：缺勤、员工关系、劳动力规划、健康安全
- **横向支撑功能**（21）：工作流、审批、通知、集成、系统管理

同时提供模块依赖关系图、典型业务流程、用户角色视图等实用内容，作为企业级 Core HR 系统功能设计与需求分析的完整参考基准。

## 1. 组织管理 (Organization Management)

### 主要功能
- 组织架构设置
- 部门/业务单元管理
- 职位管理
- 汇报关系

### 说明
负责整个企业的组织结构定义，包括层级关系、部门划分、职位设置等基础组织框架。

---

## 2. 人员管理 (Workforce Administration)

### 主要功能
- 员工主数据
- 个人信息维护
- 雇佣记录
- 员工状态管理

### 说明
管理员工的基本信息和雇佣关系，是人力资源管理的核心数据基础。

---

## 3. 职位管理 (Position Management)

### 主要功能
- 职位创建与维护
- 职位预算控制
- 职位层级结构

### 说明
以职位为中心的管理模式，支持职位预算、编制控制等企业人力资源规划功能。

---

## 4. 人事管理 (Personnel Management)

### 主要功能
- 入职 (Hire)
- 转岗 (Transfer)
- 晋升 (Promotion)
- 离职 (Termination)
- 调薪 (Compensation Change)

### 说明
处理员工在职期间的各类人事变动，是 HR 日常操作的核心模块。

---

## 5. 工作信息 (Job Data)

### 主要功能
- 职务 (Job Code)
- 职级 (Grade)
- 薪酬计划
- 工作地点

### 说明
定义和管理职务、职级等工作相关的分类信息，为人事决策提供标准化依据。

## 11. 招聘管理 (Recruiting / Talent Acquisition)

### 主要功能
- **职位需求管理**
  - 招聘需求申请与审批
  - 职位发布与管理
  - 招聘预算控制
- **候选人管理**
  - 候选人信息录入与追踪
  - 简历解析与筛选
  - 候选人库管理
- **招聘流程管理**
  - 面试安排与反馈
  - 评估表与评分
  - 面试官管理
- **录用管理**
  - 背景调查
  - 录用通知 (Offer Letter)
  - 入职准备 (Onboarding Preparation)
- **招聘渠道管理**
  - 招聘网站集成
  - 校园招聘管理
  - 内部推荐管理
  - 猎头/中介管理

### 说明
覆盖从人力需求识别到候选人录用的全流程管理，是人才获取的核心模块，与入职流程紧密衔接。

## 12. 绩效管理 (Performance Management)

### 主要功能
- **目标管理**
  - 目标设定 (Goal Setting)
  - 目标对齐与分解
  - OKR/KPI 管理
  - 目标进度追踪
- **绩效评估**
  - 评估周期配置（年度/半年度/季度）
  - 评估表设计与配置
  - 自评、上级评、同级评、下级评
  - 360 度反馈
- **绩效校准**
  - 校准会议管理 (Calibration Session)
  - 绩效分布控制（强制分布/正态分布）
  - 绩效等级调整
- **绩效改进**
  - 绩效改进计划 (PIP - Performance Improvement Plan)
  - 辅导与发展计划
  - 跟踪与反馈
- **绩效结果应用**
  - 绩效与薪酬挂钩
  - 绩效与晋升关联
  - 绩效历史记录

### 说明
支持完整的绩效管理循环（目标设定 → 过程跟踪 → 绩效评估 → 结果应用 → 改进提升），是人才发展的关键驱动力。

## 模块依赖关系

### 核心数据流

```
组织管理 (1) ──┬──> 职位管理 (3) ──> 招聘管理 (11) ──> 人员管理 (2)
                │                                              │
                │                                              ├──> 人事管理 (4)
                │                                              │
                └──> 工作信息 (5) <─────────────────────────────┘
                            │
                            ├──> 薪酬管理 (6) ──> 薪资计算 (15)
                            │
                            ├──> 绩效管理 (12) ──┬──> 人才管理 (14)
                            │                    │
                            └──> 培训与发展 (13) ─┘

时间与考勤 (8) ──> 缺勤管理 (17) ──> 薪资计算 (15)

福利管理 (7) ──> 薪资计算 (15)

合规管理 (16) ◄── [监控所有模块]

报表与分析 (10) ◄── [汇总所有模块数据]
```

### 关键依赖说明

1. **组织管理** 是基础，所有模块都依赖组织结构数据
2. **人员管理** 是核心，员工主数据被所有业务模块引用
3. **工作信息** 提供职务、职级等分类数据，影响薪酬、绩效、培训等模块
4. **人才发展链路**：绩效 ↔ 培训 ↔ 人才管理 形成闭环
5. **薪资计算** 依赖时间考勤、缺勤、薪酬、福利等多个模块的数据
6. **合规管理** 横向监控所有业务操作，确保合规性

## 典型业务流程

### 1. 员工全生命周期管理

```
招聘需求 (11) → 候选人筛选 (11) → 面试评估 (11) → 发放Offer (11)
    ↓
入职办理 (4) → 试用期管理 (4) → 转正 (4)
    ↓
在职管理：
  - 日常考勤 (8/17)
  - 绩效评估 (12)
  - 培训发展 (13)
  - 薪酬调整 (4/6)
  - 职位变动 (4)
    ↓
离职管理 (4) → 离职面谈 (18) → 离职结算 (15) → 档案归档 (16)
```

### 2. 人才发展链路

```
招聘入职 (11) → 新员工培训 (13) → 岗位培训 (13)
    ↓
绩效目标设定 (12) → 过程辅导 (12) → 绩效评估 (12)
    ↓
绩效结果分析 → 高潜识别 (14) / 绩效改进 (12)
    ↓
高潜人才 → 发展计划 (14) → 专项培训 (13) → 晋升/轮岗 (4)
    ↓
关键岗位 → 继任计划 (14) → 继任者培养 (13/14)
```

### 3. 月度薪资处理流程

```
考勤数据确认 (8) → 缺勤数据审批 (17) → 绩效奖金核算 (6/12)
    ↓
薪资数据汇总 (15) → 特殊调整录入 (15) → 社保公积金计算 (15)
    ↓
个税计算 (15) → 薪资审核 (15) → 生成工资单 (15)
    ↓
银行支付文件 (15) → 工资发放 (15) → 工资条推送 (9)
```

### 4. 年度人才盘点流程

```
确定盘点范围 (14) → 绩效数据收集 (12) → 潜力评估 (14)
    ↓
九宫格制作 (14) → 校准会议 (14) → 人才分类 (14)
    ↓
高潜人才：发展计划 (14) + 继任安排 (14)
低绩效人才：改进计划 (12) / 淘汰 (4)
核心骨干：保留激励 (6/7)
```
