# 前端效果验证指南

## 🖥️ 桌面端验证步骤

1. **打开浏览器**，访问 http://localhost:3000/employees
2. **查验核心功能**：
   - 页面顶部：4个统计卡片（总员工、在职、待入职、部门数）
   - 数据可视化：部门分布饼图 + 员工状态柱状图
   - 智能筛选：搜索框、部门筛选、预设方案
   - 视图切换：表格视图 ↔ 卡片视图

## 📱 移动端验证步骤

1. **打开浏览器开发者工具** (F12)
2. **点击设备模拟图标** (通常在左上角)
3. **选择设备**：iPhone 12 Pro 或 Galaxy S21
4. **验证响应式效果**：
   - 统计卡片：4列变为1列堆叠
   - 按钮文字：部分文字在小屏幕隐藏，只显示图标
   - 筛选工具：垂直布局，更紧凑的间距
   - 卡片视图：3列变为1列，头像变小

## 🎨 视觉设计验证

1. **Workday企业风格**：
   - 主色调：专业蓝色 (#0073e6)
   - 卡片阴影：微妙的投影效果
   - 趋势指示器：绿色上升、红色下降

2. **交互体验**：
   - 悬停效果：卡片轻微上移 + 阴影增强
   - 点击反馈：按钮缩放效果
   - 加载状态：骨架屏动画

## 💻 命令行验证

如果无法使用浏览器，可以用curl测试：

```bash
# 测试主页
curl -s http://localhost:3000 | grep -o "<title>.*</title>"

# 测试员工页面
curl -s http://localhost:3000/employees | head -20

# 测试组件页面
curl -s http://localhost:3000/test-new-components | grep -i "component"
```

## 🛠️ 开发服务器管理

```bash
# 查看服务器状态
ps aux | grep "next dev"

# 查看服务器日志
tail -f dev.log

# 停止服务器
kill $(cat dev.pid)

# 重启服务器
npm run dev
```
