## 🎯 标准分类

### 按适用范围分类
- **全局标准**: 适用于整个项目的通用规范
- **模块标准**: 特定功能模块的专用规范  
- **角色标准**: 针对特定角色(开发、测试、运维)的规范

### 按强制性分类
- **强制性规范**: 必须严格遵守的规则
- **推荐性标准**: 建议遵循的最佳实践
- **参考性指南**: 提供参考的操作建议
